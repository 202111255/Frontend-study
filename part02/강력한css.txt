<분리된 css 연결하기>
<link rel="stylesheet" href=./index.css>
rel: 해당 태그로 연결시켜줄 파일과 어떤 관계인지 지정
href: 연결할 파일의 경로를 지정


<선택자>
1. 태그 선택자 (ex. div)
tag {

}
2. id 선택자
#id {

}
3. class 선택자
.class {

}
4. 자손 선택자
.parent .child {
    //반드시 한칸 띄워주어야 함
}
5. 다중 선택자
.class#id {
    //두 조건을 모두 만족하는 태그에만 스타일 적용
    //하나만 뽑아오기 힘들 때 유용하게 사용!!
}


<폰트 기본속성>
- font-size : 폰트의 사이즈를 조절하는 속성
- font-weight : 폰트의 굵기를 조절하는 속성
- font-style : 글자의 모양을 정하는 속성으로 기울임 여부를 정함
- test-decoration : 텍스트에 장식용 선을 추가
- color` : 폰트의 색상을 변경해주는 속성
